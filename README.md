### Current Year

App to return a current year.

#### Usage

Local or [web](http://currentyear.herokuapp.com/)
``` bash
/       # return a html
/json   # return a json
```

#### Dependencies to run
``` bash
$ bundle install
# gem install sinatra
# gem install json
```

#### Run
``` bash
$ ruby web.rb
# access localhost:4567
```
